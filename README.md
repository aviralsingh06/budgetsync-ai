# BudgetSync AI

AI-Powered Finance & Budget Management Platform built during a Hackathon.

## Overview

BudgetSync AI helps users manage budgets, track expenses, monitor savings, analyze spending behavior, and receive AI-powered financial recommendations.

## Features

- Secure Role-Based Login
- Financial Dashboard
- Live Analytics Dashboard
- Voice Command Support
- AI Financial Assistant
- Budget Tracking
- Savings Monitoring
- Financial Health Score
- Audit & Reports

## Tech Stack

- React
- TypeScript
- Vite
- Gemini API
- GitHub
- Vercel

## 🎯 Problem Statement

Managing budgets manually is difficult and often leads to overspending, poor savings planning, and lack of financial awareness.

BudgetSync AI solves this problem by combining:

- Real-time finance tracking
- AI-powered recommendations
- Live analytics visualization
- Smart spending insights
